# note Analytics

note.com のクリエイター向けアナリティクス Chrome 拡張です。フォロワー増減、記事別スキ推移をリアルタイムで可視化します。

## 機能

- **フォロワーの日次増減が一目でわかる** — 日ごとの増加数をグラフ表示
- **スキの日次増加を可視化** — どの記事に今日スキがついたかが一目瞭然
- **記事ごとのスキ推移** — 折れ線グラフで各記事のパフォーマンスを追跡
- **スキランキング** — 1 日あたりのスキ増加数で記事をランキング化
- **データ管理** — ブラウザのローカルストレージに保存、いつでもデータをインポート/エクスポート

## インストール手順

### 1. リポジトリをクローン
```bash
git clone https://github.com/masay-konno/note-analytics.git
cd note-analytics
```

### 2. Chrome 拡張として登録

1. Chrome を開き、`chrome://extensions/` にアクセス
2. 右上の **開発者モード** をオンにする
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. クローンしたフォルダ（`note-analytics`）を選択
5. 拡張機能が登録されます

### 3. note.com にアクセス

note.com のダッシュボードにアクセスすると、Chrome 拡張のアイコンがツールバーに表示されます。

## 使い方

### ポップアップ画面を開く

Chrome ツールバーの拡張機能アイコンをクリックするとポップアップが開きます。

**主な表示内容：**
- **フォロワー数推移グラフ** — 過去のフォロワー数変化
- **スキ増加数グラフ** — 1 日あたりのスキ増加数
- **記事別スキランキング** — 各記事のスキ付与速度をランキング表示

### 設定画面を使う

1. ポップアップの **⚙️ 設定** ボタンをクリック
2. 以下の操作ができます：
   - **データのインポート** — 以前エクスポートしたデータを復元
   - **データのエクスポート** — 現在のデータを JSON ファイルとして保存
   - **データをリセット** — すべての蓄積データを削除

## 仕組み

note Analytics は以下のように動作します：

1. **API からデータ取得** — note.com が画面表示に使用しているのと同じ API からフォロワー数とスキ数を定期的に取得
2. **日次スナップショット保存** — データをブラウザのローカルストレージに日ごとに保存
3. **差分計算** — 前日との差を計算してグラフ表示
4. **プライバシー保護** — ローカルストレージに保存されるため、データはあなたの PC に留まります。サーバーには送信されません

## データ保護について

- 取得できるデータはあなた自身のデータ（フォロワー数、あなたの記事のスキ数）のみです
- データはブラウザのローカルストレージに保存されます
- データはあなたの PC 外には送信されません
- 詳細は [プライバシーポリシー](./docs/privacy-policy.html) をご覧ください

## よくある質問

**Q: 他のクリエイターのデータも見られますか？**  
A: いいえ。このツールは note にログインしているあなたのデータのみを取得します。

**Q: 過去のデータも見られますか？**  
A: 拡張機能をインストールした日以降のデータのみ記録されます。

**Q: データはどこに保存されていますか？**  
A: すべて Chrome のローカルストレージに保存されます。削除する場合は、ブラウザのキャッシュ削除またはこの拡張機能をアンインストールしてください。

**Q: オフラインで使えますか？**  
A: いいえ。note.com へのアクセスが必要です。
